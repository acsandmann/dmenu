simple chooser akin to choose-gui or dmenu

### usage:
```bash
ls | ./dmenu -p "Select a file" -i -l
```

### arguments(all optional):
```bash
-p "" # prompt in search box(default is "Search")
-i # enables search icon
-xs # extra small window
-s # small window
-m # medium window(default)
-l # large window
```
